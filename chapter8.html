8.
About
this
Project

This website was created
to achieve two goals.

1.)
Provide an overview of Mondrian's life and work,
divided according to important stylistic shifts.

2.)
Demonstrate the application of Mondrian's ideas to web design.

If you want to learn more about Piet Mondrian and his art,
I highly recommend the website
Mondrian Route.

This website is fun, impeccably researched, beautifully crafted,
and full of intimate biographical details.

Although I only discovered Mondrian Route towards the end of making this website,
it was very valuable in rounding out my understanding of Mondrian's life.

For a more academic analysis of Mondrian's art,
I recommend
Michael Sciam's website.

This website is pretty dense,
with a lot of focus on Mondrian's approach to composition.

It's perhaps a bit more technical than some readers may want,
but offers a very deep analysis of Mondrian's work.

As for the application of Mondrian's aesthetic to web design,
I consider this website to be a reasonable success.

Certainly some parts of the design work better than others.

I was pretty much making things up as I went along,
experimenting with different ideas.

At the very least, I think there are some good design ideas in the mix,
and I hope to improve upon these ideas in the future.

This website exclusively uses the
web-font Montserrat.

If you want to check out Montserrat,
you can view it here on Google Fonts.

All backgrounds and icons were designed by myself,
using plain old CSS for backgrounds and SVG for icons.
